## Color Tokens

### Light Mode
| Token | Hex | Role |
|-------|-----|------|
| `--bg` | `#FAF7ED` | Cream/Ivory main background |
| `--surface` | `#FFFFFF` | Cards, elevated surfaces |
| `--fg` | `#083D48` | Primary text (petroleum blue) |
| `--muted` | `#5A7A82` | Secondary text |
| `--border` | `#D6E0E3` | Subtle borders |
| `--accent` | `#EC9960` | Soft orange for CTAs and highlights |
| `--primary` | `#0D5C6F` | Deep teal for links, icons, branding |

### Dark Mode

## Typography
- **Display:** 'Plus Jakarta Sans', system-ui, sans-serif (700, 800)
- **Body:** 'Plus Jakarta Sans', system-ui, sans-serif (400, 500, 600)
- **Mono:** 'JetBrains Mono', ui-monospace, monospace

## Logo Contrast
- On light surfaces, use the full-color logo directly over ivory or light backgrounds.
- On dark surfaces, switch the logo to white with a CSS filter so it stays legible without a white plate.

## Layout Posture
- Generous whitespace (gap-16, py-32)
- Rounded corners (rounded-3xl = 24px)
- Asymmetric bento grid compositions
- Ultra-fine kinetic lines (1px strokes)
- One accent (orange) used at most 2x per section
